# Decisions

Lightweight decision notes captured during Claude sessions.

## Relationship to ADRs

Full Architecture Decision Records live in `docs/adr/`. This folder is for
smaller, session-level decisions that may not warrant a formal ADR but are
still worth remembering (e.g., "chose X library over Y because...").

Decision notes will be created in Phase 2, either manually or by a future
PostToolUse hook that detects ADR-worthy file changes.
